# My Birthday Surprise 💖

A magical birthday surprise for Myra, featuring:
- Personalized messages
- Photo gallery
- Background music ("Pal")
- Special voice recording message
- Confetti and heart animations

## How to View
1. Open `tp.html` in any web browser.
2. Click **"Scan My Surprise"** to start the magic!

## Project Structure
- `tp.html`: The main website file.
- `audio/`: Contains the background music and voice recording.
- `images/`: Contains the photos used in the gallery.
